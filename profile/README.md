# MC Imagineers

**A 1:1 scale recreation of Walt Disney World's Magic Kingdom, built block by block in Minecraft: Java Edition.**

[**mcimagineers.com**](https://mcimagineers.com) · [Discord](https://discord.gg/TYVQBfDPYX) · [YouTube](https://www.youtube.com/channel/UCpzmcJhF4vBsKTwnJd-RnAw) · [TikTok](https://tiktok.com/@mc_imagineers)

---

MC Imagineers is a volunteer-driven project bringing Magic Kingdom into Minecraft at true 1:1 scale — custom ride systems, synchronized time and weather, and hundreds of hand-built 3D models.

-  **[Visit the site](https://mcimagineers.com)** — progress updates, the attraction Navigator, and more
-  **[Foundry](https://mcimagineers.com/foundry/)** — our free browser tool for planning to-scale Minecraft builds
  
-  **[Join the team](https://mcimagineers.com/apply/)** — we're always looking for builders, developers, and Guest Experience volunteers

*MC Imagineers is an independent, fan-made project and is not affiliated with, endorsed by, or connected to The Walt Disney Company, Mojang Studios, or Microsoft.*
